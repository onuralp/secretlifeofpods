What is a Pod?

Running instances of applications
  • "Smallest schedulable unit"
  • One or more containers that share the namespace/are tightly coupled
  • Ephemeral!
  • Namespace?
    1. Shared memory (SHM)
    2. Volumes
    3. IP
    4. hostname

Everything's just Pod templates

               
apiVersion: v1
kind: ReplicationController
metadata:
  name: ptlcdebug
spec:
  replicas: 3
  selector:
    app: ptlcdebug
  template:
    metadata:
      name: ptlcdebug
      labels:
        app: ptlcdebug
    spec:
      containers:
      - command:
        - sleep
        - "86400"
        image: alpine:3.4
        name: alpine
